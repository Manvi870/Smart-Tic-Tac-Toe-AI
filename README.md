# Smart-Tic-Tac-Toe-AI

An advanced Tic Tac Toe game built using Python and Tkinter.

## Features

- Medium AI Opponent
- Player Name Input
- Score Tracking System
- Statistics Panel
- Live Timer
- Dark / Light Theme Toggle
- Winner Highlighting
- Winner Animation
- Sound Effects
- Responsive GUI Design

## Technologies Used

- Python
- Tkinter
